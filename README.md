#Snake Game (Unity, C#)

##Overview
This project is a Snake-style game developed using Unity and C#.  
It demonstrates core game development concepts including real-time movement, collision detection, and game state management.

The project was independently designed and implemented from scratch, showcasing my ability to build a complete interactive system.

---

##Features
- Real-time snake movement
- Collision detection (walls & self)
- Score tracking system
- Game loop and input handling
- Smooth gameplay mechanics

---

##Tech Stack
